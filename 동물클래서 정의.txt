클래서 정의

1. 동물 클래서 정의
	- Animal
	- 생성자(디폴트, 매개변수 1개 : 동물이름, 크기(30 ~ 200)와 속도(0)는 디폴트로 설정)
	- 오버로딩 된 생성자 (매개변수 2개 :  동물이름, 크기)

2. 속성 정의
	- 동물 이름(String)
	- 크기 : 몸무게(int)
	- 달리는 속도(int)

3. 메서드 정의
	- 달리는 속도 증가(기본증가 10, 주어진 값 증가) : 0 ~ 100 사이만 가능
	- 달리는 속도 감소(기본감소 10, 주어진 값 감소) : 0 ~ 100 사이만 가능
	- get, set 정의(각각)

4. 테스트 클래스 생성하여 테스트(사자 2건, 호랑이, 기린, 코끼리....)
